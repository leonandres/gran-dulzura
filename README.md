# Gran Dulzura 🍰

Landing page responsive para una pastelería artesanal desarrollada con HTML, CSS y Bootstrap 5 y JavaScript.

El web ofrece una presentación moderna y visual de productos de repostería, incluyendo catálogo, contacto y pedidos por WhatsApp.

---

## | Características

- Diseño responsive
- Navbar adaptable para móviles
- Hero principal con imagen destacada
- Cards de productos
- Badges para productos destacados
- Botones de pedido por WhatsApp
- Footer moderno con redes sociales
- Scroll suave entre secciones
- Estilos personalizados sobre Bootstrap 5

---

## | Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5.3
- Bootstrap Icons
- JavaScript
